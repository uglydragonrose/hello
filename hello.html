git clone url ::: είναι η εντολή που γράφουμε για να κατεβάσουμε το repository απο το github στον υπολογιστή μας

μετά απο κάθε αλλαγή που κάνουμε πατάμε ctr+s για να σώσουμε τις αλλαγές που κάναμε στον υπολογιστή μας

έτσι ώστε μετά στο τερματικό μας γράφωντας για windows λειτουργικό start το όνομα του αρχείου μας να ανοίξουμε το αρχείο μας

με τις νέες αλλαγές που κάναμε στην περίπτωση μας το συγκεκριμένο αρχείο html

Με την εντολή touch hello.html στο τερματικό μας δημειουργούμε το html αρχείο git add filename είναι η εντολή για να σώσουμε το αρχείο στο repository git commit -m "message" είναι η επόμενη εντολή για να σώσουμε το αρχείο git push είναι η τελική εντολή για να ολοκληρωθεί η ενημέρωση του repository git commit -am "message" είναι η εντολή για να ανεβάσουμε όλους τους φακέλους που έχουμε αλλάξει
